# Portfolio (React + TSX + Vite)

## Jalankan

1. Masuk folder:
   - `cd projectku`
2. Install dependency:
   - `npm install`
3. Dev server:
   - `npm run dev`
4. Build:
   - `npm run build`
5. Preview build:
   - `npm run preview`

## Edit konten

- Data profil & proyek: `src/content/profile.ts`
- Layout utama: `src/App.tsx`
- Styling global: `src/styles.css`
